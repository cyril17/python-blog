## Plog
由 Python+Flask 开发的个人博客程序

技术栈
* 后端
  - Python
  - Flask 模板引擎
  - MongoDB 数据库
  - Redis RestFul API 高速缓存数据库以及页面缓存
* 前端
  - Bootstrap CSS 框架
  - LESS CSS预处理编译器
  - jQuery
  - Gulp/Webpack JS模块化以及静态资源自动化合并压缩发布
